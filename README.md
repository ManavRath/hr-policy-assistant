# HR Policy Assistant

## Overview
This project is an AI-powered HR assistant that answers employee queries using company policy documents with Retrieval-Augmented Generation (RAG).

## Features
- HR query answering
- Context-based responses (no hallucination)
- Streamlit UI
- Conversation memory

## Tech Stack
- Python
- ChromaDB
- Sentence Transformers
- Groq LLM
- Streamlit

## How to Run
pip install -r requirements.txt  
streamlit run app.py
